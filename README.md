# ESP32 Projects

This repository contains ESP32-based embedded systems projects developed using Arduino IDE.

## Projects

- ESP32 LED Blink
- ESP32 7-Segment Display Counter
- ESP32 Distance Measurement using Ultrasonic Sensor

## Repository Structure

```
ESP32-Projects
├── ESP32_LED_Blink
├── ESP32_7_Segment_Display_Counter
└── ESP32_Ultrasonic_Distance_Measurement
```

## Hardware

- ESP32 Development Board
- LEDs
- 7-Segment Display
- HC-SR04 Ultrasonic Sensor

## Software

- Arduino IDE
- Embedded C
- Wokwi Simulator
